**fssemR** is an R package for differential gene regulatory network inference.

**Methods:**
- Non-convex optimization with ADMM / augmented Lagrangian
- Structured sparsity for network estimation
- Likelihood-based estimation with convergence diagnostics

Published in *Bioinformatics* 36(1):197--204, 2020.
